# Angular 19 ToDo App ✅  
🚀 Live Demo: [abubakkar-todo-app.netlify.app](https://abubakkar-todo-app.netlify.app/)  

A **feature-rich standalone Angular 19 To-Do application** designed for efficient task management. This app works **offline** using **Angular Service Workers**, integrates a real-time **Weather API**, and features an engaging **Hangman game**. It offers a **responsive user interface** for seamless usage across all devices.  

---

## 📌 Features  
✅ **Task Management**: Add, complete, and delete tasks  
✅ **Offline Mode**: Powered by **Angular Service Workers**  
✅ **Weather API Integration**: Fetches live weather updates  
✅ **Hangman Game**: A fun bonus feature for users  
✅ **Responsive Design**: Fully optimized for desktop, tablet, and mobile  

---

## 🚀 How to Use  

### Development Server  
To start a local development server:  
```bash
ng serve
```
Navigate to `http://localhost:4200/` in your browser. The application supports **hot reloading**, so changes are reflected instantly.

### Build for Production  
For production-ready builds, run:  
```bash
ng build
```
This command generates optimized build artifacts in the `dist/` directory.

### Unit Testing  
Run unit tests using [Karma](https://karma-runner.github.io):  
```bash
ng test
```

### End-to-End Testing  
Run E2E tests (add a testing framework if not configured):  
```bash
ng e2e
```
